# Mini Journal App

A simple React + Firebase app to create, view, and edit notes with user authentication. This mini project helps you learn React state management, Firebase Authentication, and Firestore cloud storage.

## Features

- User authentication (Signup / Login / Logout)
- Create new notes
- View a list of all notes
- Edit existing notes
- Notes are stored in Firebase Firestore and tied to a specific user


## Tech Stack

React – Frontend UI

Firebase Authentication – User signup and login

Firebase Firestore – Cloud-based note storage

Vite – Development server and bundler

## Getting Started
**1. Clone the repo**
```bash
git clone https://github.com/your-username/mini-journal-app.git
cd mini-journal-app
```

**2. Install dependencies**
```bash
npm install
```

**3. Firebase Setup**

Go to Firebase Console.

Create a new project.

Enable Email/Password Authentication.

Create a Firestore database.

**4. Run the app**
```bash
npm run dev
```

Open your browser at http://localhost:5173 (or the link provided by Vite).

Signup / Login and start creating notes.
